
# Calculator Discount


## Description:

an application that is used to help users divide the amount of dues fees for orders ordered after getting a discount

## Tech Stack

**Server:** Golang, gorm v1.24.0,MySQL
**Client:** React,Tailwind


## Method
GET, POST, PATCH, DELETE
